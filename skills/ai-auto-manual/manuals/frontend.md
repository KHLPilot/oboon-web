# Frontend Manual

## Scope

- UI components
- page/layout rendering
- client-side data fetch flow

## Required Practices

1. Guard loading and error UI states.
2. Keep data-fetch side effects isolated.
3. Avoid exposing secrets in client bundle.
4. Keep accessibility labels and keyboard focus behavior.

## Output Checklist

- Mention loading/error state handling.
- Mention user-facing validation feedback.
