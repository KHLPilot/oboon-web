# OBOON 전체 기능 명세서 (완성본)

본 문서는 OBOON 서비스의 전체 기능을 **일반 사용자/상담사/관리자** 관점으로 정리하고,  
**기능 개요, 사용자 시나리오, 화면 설계, 데이터 흐름, API 연동, 예외 처리, 용어집**까지 포함한 완성형 명세서입니다.

---

# 1. 공통 기능

## [F-001] 인증/회원가입

### 기본 정보

- 기능 ID: F-001
- 기능명: 인증/회원가입
- 담당자: 기획/개발/디자인
- 우선순위: High

### 기능 개요

사용자가 계정을 생성하고 로그인/로그아웃을 수행한다.

### 사용자 시나리오

사용자가 로그인 화면에서 이메일/소셜 로그인을 시도할 때

### 화면 설계(와이어프레임)

- 로그인 폼, 소셜 로그인 버튼
- 성공/실패 토스트 표시

### 데이터 흐름

- 입력값: 이메일/비밀번호, 소셜 토큰
- 출력값: 세션 토큰, user 정보
- 처리: 인증 성공 → 세션 저장 → 프로필 확인

### API 및 연동 정보

- Supabase Auth 사용

### 예외 처리 및 에러 케이스

- 로그인 실패 시 에러 표시
- 세션 만료 시 재로그인 요구

## [F-002] 프로필/마이페이지

### 기본 정보

- 기능 ID: F-002
- 기능명: 프로필/마이페이지
- 담당자: 기획/개발/디자인
- 우선순위: High

### 기능 개요

사용자가 프로필 정보를 확인/수정하고 마이페이지 기능을 이용한다.

### 사용자 시나리오

사용자가 마이페이지에서 프로필 이미지를 변경할 때

### 화면 설계(와이어프레임)

- 프로필 이미지 영역, 업로드 버튼
- 기본정보 카드, 내 예약 카드

### 데이터 흐름

- 입력값: 아바타 이미지 파일, 닉네임
- 출력값: 업데이트된 profiles.avatar_url
- 처리: 업로드 → URL 생성 → profiles 업데이트

### API 및 연동 정보

- Supabase Storage `avatars`
- Supabase profiles 테이블 update

### 예외 처리 및 에러 케이스

- 이미지 용량 초과/타입 오류 안내
- 로그인 상태 아니면 업로드 제한

- 담당자: 기획/개발/디자인
- 우선순위: High

### 기능 개요
