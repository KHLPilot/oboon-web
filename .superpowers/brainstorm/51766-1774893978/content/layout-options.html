A — 카테고리 탭 + 히어로

B — 히어로 + 섹션 블록 스크롤

풀블리드 히어로 아래, "오분 오리지널 → 전체 보기", "일반 브리핑 → 전체 보기" 섹션을 수직 스택. 각 섹션 카드 3~4개 미리보기.

오분 오리지널
일반 브리핑

C — 검색 상단 + 히어로 + 2컬럼

검색바를 최상단에 배치 후 히어로, 히어로 아래 좌(오리지널)/우(일반) 2컬럼 미리보기. 검색이 핵심 기능일 때 적합.
