# 관리적 보안 취약점 분석 — A-01 ~ A-127

**대상**: OBOON-WEB 프로젝트 관리 체계
**평가 일자**: 2026-03-29

> 관리적 평가는 문서/정책/절차의 존재 및 이행 여부를 기준으로 한다.
> 코드 분석으로 확인 가능한 항목과 문서화 필요 항목을 구분하여 평가한다.
