## 역할 범위

관리자(Admin)는 플랫폼 전체를 관리한다.

- 상담사 계정 승인/거절
- 회사(시행/시공사) 승인/거절
- 분양 현장(offerings) 승인/관리
- 플랫폼 전체 통계/모니터링

---

## 관련 API

| API 경로 | 용도 |
|----------|------|
| `app/api/admin/**` | 관리자 승인/관리 액션 |

---

## 주의사항

- 관리자 페이지는 **인증 + 역할 검증** 필수 (`role === 'admin'`)
- RLS 정책에서 admin 역할에만 허용된 테이블 접근 多
- 관리자 액션(승인/거절)은 서버 API를 통해서만 수행 (클라이언트 직접 DB 쓰기 금지)
- DB SSOT: `docs/db/README.md`
